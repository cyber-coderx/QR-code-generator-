* For each finder pattern:

  1. Loop over a **7×7 square**.
  2. Fill cells along the **outer border** (top, bottom, left, right) as black.
  3. Fill the **3×3 inner square** in the center as black.

### 4. **Convert the input text into binary**

### 5. **Fill the remaining grid cells**

* Start iterating **row by row, column by column**.
* Skip the cells already filled by finder patterns.
* For each cell:

  1. Take the next bit from the binary string.
  2. If the bit is `1`, set the grid cell to `true` (dot present).
  3. If the bit is `0`, leave it `false` (empty).

### 6. **Check neighbors for each dot**

* This ensures:

  * Isolated dots → fully rounded.
  * Dots touching others → only the corners without neighbors are rounded.
